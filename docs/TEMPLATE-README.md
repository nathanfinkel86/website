# Music Website Template

This is a complete static website template designed for musicians, artists, and creative professionals. It features a modern, immersive design with background imagery, clean navigation, and responsive layouts.

## ğŸ¨ Template Features

- **Hero background image integration** - Full-page background with overlay and floating content
- **5 main pages** - Home, About, Music, Projects, Shows, Contact
- **Project subpages** - Individual detail pages for each project
- **Event integration** - Dandelion.events embed for automatic event listings
- **SoundCloud embeds** - Audio player integration
- **Responsive design** - Mobile-first with adaptive layouts
- **Dark theme** - Teal/cyan accent colors with dark backgrounds
- **Accessible** - Semantic HTML, skip links, proper ARIA labels
