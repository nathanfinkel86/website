Glia

A multiplayer visual programming environment for collaborative music creation. Built on CMajor and JUCE with real-time CRDT synchronization. I'm an early adopter—took it on the road for successful UK shows in December 2025.

What it is

Collaborative audio programming

Visual graph editor for building audio processors—think Google Docs for modular synthesis. Runs as VST3/AU, standalone, or web client.

How I use it

Live performance integration

Custom audio graphs in Ableton Live responding to MIDI controllers and audio input. Stress-testing the platform in real-world contexts.
